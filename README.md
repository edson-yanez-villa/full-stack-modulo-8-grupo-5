## Estructura del Proyecto

La estructura de carpetas del proyecto es la siguiente:

```plaintext
repo/
├── .env
├── Dockerfile
├── docker-compose.yml
├── backend/
│   ├── manage.py
│   ├── backend/
│   │   ├── __init__.py
│   │   ├── settings.py
│   │   ├── urls.py
│   │   └── wsgi.py
│   └── pharmacy/
│       ├── __init__.py
│       ├── admin.py
│       ├── apps.py
│       ├── models.py
│       ├── serializers.py
│       ├── tests.py
│       ├── urls.py
│       └── views.py
└── requirements.txt
```
